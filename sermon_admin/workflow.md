# 설교 영상 행정 제출용 표준 워크플로우 (Sermon-Centered)

## 1단계. 원본 설교 영상 확보 (기록 보존)

* 예배 실황 영상 (카메라 / 스트리밍 녹화 / OBS 녹화본 등)
* 원본 영상은 **편집하지 않고 보관**
* 파일명 규칙 예시

```
원본_주일예배_2026-01-12_오전예배.wmv
```

📌 목적

* 설교 내용의 **원문 증거성 보존**
* 오해·분쟁·행정 요청 발생 시 근거 자료

---

## 2단계. 설교 핵심 구간 지정

### 주요 지정 기준

* 설교 시작 시점
* 핵심 메시지 구간 (예화, 적용, 사회적 발언 포함)
* 문제 소지·민감 발언 구간 (행정 대응 대비)
* 결론 및 기도

📌 시간 구간 예시

```
00:32:10 ~ 00:41:55  (핵심 본문 해설)
00:48:03 ~ 00:52:40  (적용 및 사회적 메시지)
```

📌 포인트

* **전체 예배가 아닌 ‘설교 중심’ 요약**
* 행정 담당자가 짧은 시간에 맥락 파악 가능

---

## 3단계. 설교 요약 영상 자동 생성

* 지정 구간만 추출
* 시간 순서대로 병합
* 불필요한 침묵·찬양·광고 제거

📌 결과물 목적

* 3~10분 이내 설교 핵심 요약 영상

📌 파일명

```
설교요약_주일예배_2026-01-12.mp4
```

---

## 4단계. 자동 자막 생성 (설교 특화)

* 설교 음성을 기준으로 STT 수행
* 문장 단위 자막 생성

📌 설교 영상 자막 기준

* 성경 인용은 줄 분리
* 긴 문장은 2줄 이내 자동 분할
* 기도·묵상 구간은 자막 생략 가능

📌 자막 예시

```
"오늘 본문에서 바울은
믿음의 자유와 책임을 함께 말합니다."
```

📌 결과물

* SRT (행정·문서 제출용)
* 자막 포함 영상 (시청용)

---

## 5단계. 행정·기관 제출용 패키징

### ① 제출용 영상

* 자막 포함 mp4
* 표준 코덱(H.264 + AAC)

### ② 설교 자막 원문

* 텍스트 문서 또는 SRT
* 인용·검토·기록용

### ③ 설교 개요서 (1페이지)

* 설교 일시
* 설교자
* 본문
* 핵심 요지
* 영상 구간 설명

---

## 6단계. 제출 대상별 활용

* 노회·총회: 설교 검토, 신학적 소명
* 교육청·지자체: 민원·행사 협력 자료
* 후원기관: 사역 보고
* 내부 기록: 설교 아카이빙

📌 필수 문구

> 본 영상은 이해를 돕기 위한 요약본이며
> 전체 설교 원본은 별도 보관되어 있습니다.
