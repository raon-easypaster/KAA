# 설교 영상 자동화 프로그램 구현 계획 (Sermon Automation Implementation Plan)

## 목표
설교 영상 행정 제출 표준 워크플로우를 자동화하는 Python 스크립트를 개발합니다.
사용자가 원본 영상과 시간 구간을 입력하면, 자동으로 요약 영상을 생성하고 폴더 구조를 갖추어 정리합니다.

## 기술 스택
- **Language**: Python 3.12+
- **Video Processing**: `moviepy` (FFmpeg wrapper, 바이너리 관리 용이)
- **Downloading**: `yt-dlp` (YouTube 영상 다운로드)
- **GUI**: `streamlit` (웹 기반 대화형 인터페이스)
- **File System**: `pathlib`, `shutil`

## 구현 기능

### 1. `sermon_manager.py`
- **구간 추출 및 병합**: 설정된 시간 구간(Start~End)을 잘라내어 하나의 영상으로 병합합니다.
- **폴더 구조 생성**: 
  - `원본`, `설교요약`, `자막`, `행정제출`, `설교개요` 폴더 자동 생성.
- **결과 리포트**: 
  - 처리 결과 및 파일 위치를 한글로 출력합니다.

### 2. 자막 생성 (Subtitle) - *Optional/Mock for now*
- `ffmpeg`가 시스템에 설치되지 않은 환경을 고려하여, 이번 단계에서는 자막 생성 기능을 **Mock(가상 실행)** 처리하거나, `moviepy`를 통해 오디오만 추출하는 기능까지만 구현하고 "자막 생성 필요" 파일(Dummy SRT)을 생성하여 워크플로우를 완결 짓습니다.

### 3. YouTube 다운로드 (New)
- 사용자가 YouTube 링크를 입력하면 `yt-dlp`를 사용하여 원본 폴더에 고화질로 다운로드합니다.
- 다운로드 완료 후 해당 파일을 '원본 영상'으로 간주하여 나머지 워크플로우를 진행합니다.

### 4. GUI (Streamlit)
- **`app.py`**: Streamlit을 이용한 웹 인터페이스 제공.
- **기능**:
    1. **입력**: YouTube URL 입력 또는 로컬 파일 선택.
    2. **프리뷰**: 영상을 로드하여 재생.
    3. **구간 설정**: 슬라이더/입력창을 통해 시작/종료 시간(N개 구간)을 직관적으로 설정.
    4. **실행**: '작업 시작' 버튼으로 백그라운드 처리.
    5. **결과**: 처리된 영상 다운로드 링크 및 경로 표시.

## 워크플로우 (Execution Flow)

1. **입력**: 비디오 파일 경로 (없을 경우 테스트용 더미 파일 생성) 또는 **YouTube URL**
   - 시작 시 입력 모드 선택 ( 1: 테스트 / 2: YouTube 다운로드)
   - **GUI 사용 시**: 화면에서 직접 설정
2. **처리**:
   - (YouTube 선택 시) 영상 다운로드 -> 원본 폴더 저장
   - 폴더 생성
   - 영상 자르기 및 붙이기
   - 결과 파일 저장
3. **출력**: "작업 완료" 메시지 및 폴더 경로 안내 (한글)

## 사용자 리뷰 필요 사항
- `FFmpeg`가 시스템에 설치되어 있지 않아 `moviepy`의 내장 바이너리를 사용합니다. 
- 실제 무거운 영상 처리는 시간이 걸릴 수 있으므로, 데모 실행을 위해 **5초짜리 컬러 패턴 테스트 영상**을 생성하여 전체 프로세스가 잘 돌아가는지 시연하겠습니다.
