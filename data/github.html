제미나이로 만든 콘텐츠를 웹사이트로 변환하고 깃허브에 무료로 올리는 15단계

Tutorial

배포 프로세스 시각화

아래의 단계를 따라하면 누구나 자신만의 웹페이지를 가질 수 있습니다.

01 준비하기 (제미나이로 코드 얻기)

Step Step 1

제미나이 도구에서 Canvas 플러그인을 선택하고, "내용을 깃허브에 사용할 인포그래픽 HTML로 만들어줘"라고 구체적으로 요청하세요!

명령어 입력
Step Step 2

제미나이가 문서를 작성 완료하면, 우측 상단의 복사 버튼을 눌러 코드를 전체 복사하세요.

코드 복사
Step Step 3

복사한 코드를 바탕화면 메모장에 붙여넣고 파일 이름을 index.html로 저장해 둡니다. (선택사항: 저장 없이 깃허브에 바로 붙여넣기도 가능합니다.)

파일 임시 저장

02 깃허브 저장소 및 업로드

  • 4
    깃허브 로그인 후 New Repository 버튼을 눌러 새 저장소를 생성하세요.
  • 5
    Add file 버튼을 누르고 다음 2가지 방법 중 하나를 선택하세요:
    • Upload files: 내 컴퓨터에 미리 저장해둔 index.html 파일을 마우스로 끌어서 올리기
    • Create new file: 파일경로명에 index.html라고 적고 아까 복사해둔 코드를 본문에 붙여넣기
  • 6
    작업 완료 후 하단의 초록색 Commit changes 버튼을 클릭하고, 뜨는 확인 창에서 한 번 더 Commit changes를 클릭합니다.
GitHub Repository Upload

03 최종 웹사이트 배포

7 페이지 설정 (Pages 연동)

  • 상단 Settings 메뉴 진입
  • 좌측 Pages 탭 클릭
  • Source Branch를 main으로 선택 후 Save

8 배포 완료 및 확인

코드를 변경할 때마다 상단의 Actions 탭에서 자동 실행이 진행됩니다. 실행이 완료되어 초록색 체크(✔) 표시가 뜨면 내 사이트가 완성(수정)됩니다!

완성되면 상단 배너 또는 Pages 메뉴에서 Visit site 버튼을 클릭하여 확인하세요.
