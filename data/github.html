배포 프로세스 시각화

아래의 단계를 따라하면 누구나 자신만의 웹페이지를 가질 수 있습니다.

01 제미나이에서 HTML 생성 및 저장

1~3단계: 콘텐츠 변환

제미나이 도구에서 Canvas를 선택하고 내용을 입력하세요. "깃허브용 HTML" 요청이 핵심입니다.

Gemini Canvas 활성화
4~5단계: 로컬 파일 저장

코드를 복사하여 메모장에 붙여넣은 후 반드시 index.html로 저장해야 첫 페이지로 인식됩니다.

index.html 파일 생성

02 깃허브 저장소 및 업로드

  • 07
    깃허브 로그인 후 New Repository 생성
  • 09
    Add file > Upload files 클릭하여 index.html 업로드
  • 10
    하단의 녹색 Commit changes 버튼 클릭
GitHub Repository Upload

03 최종 웹사이트 배포

페이지 설정 (11~13단계)

  • Settings 메뉴 접속
  • 좌측 Pages 탭 클릭
  • Branch를 main으로 저장

배포 완료 (14~15단계)

잠시 후 나타나는 상단 배너의 Visit site 버튼을 클릭하세요. 전 세계 누구나 접속 가능한 주소가 생성됩니다!
